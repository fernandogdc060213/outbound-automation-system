# Outbound Automation System

🚀 Automated outbound system for logistics operations

## What it does
- Sends automated emails using Outlook
- Tracks replies and updates CRM
- Manages follow-ups automatically

## Why I built it
To reduce manual prospecting work and scale outbound operations.

## Tech Stack
- PowerShell
- Excel (CRM)
- Outlook API

## Outcome
- Reduced manual workload
- Scaled outreach to hundreds of emails per day
